# 自定义题库原型

这是一个基于纯 HTML/CSS/JS 的移动端优先原型，实现了自定义题库的核心功能。

## 功能列表

### 1. 首页 (Home)
- **做题模块**:
  - **随机模式**: 点击进入随机做题。
  - **知识树模式**: 点击打开层级目录，支持展开/折叠，二级/三级节点可直接开始做题。
- **复习模块**: 显示今日待复习数量，点击开始复习。
- **记录模块**: 顶部卡片显示今日学习率和复习率。
- **新增知识体系**: 提供新增知识树和新增题目的入口。

### 2. 我的中心 (My Center)
- **错题本**: 查看错题记录。
- **数据图表**:
  - 做题复习曲线 (折线图)
  - 新增知识点趋势 (柱状图)
- **未掌握知识**: 展示错题率高的知识点列表。

## 如何运行
直接在浏览器中打开 `index.html` 文件即可预览。
推荐使用移动端模式 (F12 -> Toggle Device Toolbar) 获得最佳体验。

## 技术栈
- HTML5 / CSS3 / Vanilla JavaScript
- Font Awesome (图标)
- Chart.js (图表)
