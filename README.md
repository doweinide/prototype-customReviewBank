# 智库 AI - 自定义题库系统

这是一个基于纯 HTML/CSS/JS 的双端（Web + Mobile）原型系统，实现了自定义题库、知识管理与智能练习的核心功能。

## 🖥️ Web 端 (桌面版)

专为 1080p 大屏优化的现代化仪表盘，提供高效的知识管理与数据分析体验。

### 核心功能

#### 1. 首页概览 (Dashboard)
- **沉浸式布局**: 针对桌面端优化的 `One-Screen` 设计，无全局滚动条，核心信息一目了然。
- **数据可视化**:
  - **多维趋势图**: 同时展示「复习量」、「新学」和「错题」三条曲线，精准追踪学习动态。
  - **题库分布**: 环形图展示各知识点占比。
- **快捷入口**: 随机测验、专项练习、错题本、记笔记等常用功能一键直达。

#### 2. 知识体系管理 (Knowledge)
- **树形结构**: 支持无限层级的知识点管理（增删改查）。
- **题目录入**: 支持单选、多选等多种题型录入，并提供 AI 辅助生成题目功能（模拟）。
- **批量导入**: 支持 JSON 格式批量导入题目数据。

#### 3. 智能笔记 (Notes)
- **双栏编辑**: 左侧列表，右侧实时编辑，支持 Markdown 风格。
- **题目关联**: 笔记可直接关联相关题目，实现“学练结合”。

#### 4. 个人中心与设置 (User Center)
- **分栏式布局**: 统一管理个人资料、学习统计、系统设置与账号安全。
- **能力雷达**: 大尺寸雷达图展示六维能力模型（基础、原理、实战等）。
- **学习热力图**: GitHub 风格的每日学习热力展示。
- **系统设置**: 支持音效开关、数据备份与恢复（JSON 格式）。

### 技术栈 (Web)
- **Tailwind CSS**: 原子化 CSS 框架，构建现代化 UI。
- **Chart.js**: 强大的数据可视化图表库。
- **Font Awesome**: 丰富的图标库。
- **Inter Font**: 优质的屏幕阅读字体。

## 📱 Mobile 端 (移动版)

轻量级移动端原型，专注于碎片化时间的刷题与复习。

### 功能列表

#### 1. 首页 (Home)
- **做题模块**:
  - **随机模式**: 点击进入随机做题。
  - **知识树模式**: 点击打开层级目录，支持展开/折叠。
- **复习模块**: 显示今日待复习数量。

#### 2. 我的中心 (My Center)
- **错题本**: 查看错题记录。
- **数据图表**: 基础的做题复习曲线。

### 技术栈 (Mobile)
- HTML5 / CSS3 / Vanilla JavaScript
- Font Awesome (图标)
- Chart.js (图表)

---

## 🚀 如何运行

### Web 端
直接在浏览器中打开 `web/index.html` 文件即可体验完整功能。

### Mobile 端
在浏览器中打开 `mobile/index.html`，并切换至移动端模式 (F12 -> Toggle Device Toolbar) 获得最佳体验。
